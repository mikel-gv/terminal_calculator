A robust and simple console-based calculator designed to handle basic and advanced arithmetic operations without crashing.

## Features
- **Continuous Loop:** Perform multiple operations without restarting the program.
- **Error Handling:** Built-in `try-except` blocks to catch non-numeric inputs and division-by-zero errors.
- **Input Validation:** Ensures the user provides valid data before proceeding.
- **User-Friendly:** Clear prompts and a quick "Exit" command to close the application at any time.

## Supported Operations
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Power (`**`)
- Modulo (`%`)
- Floor Division (`//`)
